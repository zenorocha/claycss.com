So Clay is the new Lexicon?

Not exactly. There's always been a distinction between the experience language Lexicon Experience Language, and the web implementation Lexicon.

As you can see, naming them like this didn't help with the distinction. So Lexicon, the web implementation, has been renamed to Clay.
